Mini Shop 

⚙️ Tech Stack<br>
 . React + VITE<br>
 . TypeScript <br>
 . JavaScript<br>
 . TailwindCss<br>
 . i18next<br>

🔋 Features<br>
👉 Landing Page: homepage that introduces products.<br>
👉Product Details Page: Detailed product info and images.<br>

🤸 Quick Start<br>
Git<br>
npm (Node Package Manager)<br>

Installation<br>
Install the project dependencies using npm:<br>
👉npm install<br>

Running the Project:<br>
👉npm run dev<br>
